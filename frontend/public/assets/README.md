# Tree Assets

This folder contains images for the Christmas tree feature.

## Tree Images

Place tree images for different levels in `tree/`:
- `tree-level-1.png` - Level 1 tree (smallest)
- `tree-level-2.png` - Level 2 tree
- `tree-level-3.png` - Level 3 tree
- `tree-level-4.png` - Level 4 tree
- `tree-level-5.png` - Level 5 tree (largest)

**Recommended specs:**
- Format: PNG with transparency
- Size: 400x600px to 500x700px
- Background: Transparent or white
- Style: Realistic Christmas tree illustration

## Decoration Images

Place decoration images in `decorations/`:
- `ornament.png` - Christmas ornament
- `star.png` - Star decoration
- `light.png` - Christmas light
- `garland.png` - Garland decoration
- `gift.png` - Gift box
- `snowflake.png` - Snowflake

**Recommended specs:**
- Format: PNG with transparency
- Size: 60x60px to 80x80px
- Background: Transparent
- Style: Realistic or illustrated decorations

## Image Sources

You can find free Christmas tree and decoration images from:
- [Unsplash](https://unsplash.com/s/photos/christmas-tree)
- [Pexels](https://www.pexels.com/search/christmas-tree/)
- [Pixabay](https://pixabay.com/images/search/christmas%20tree/)
- [Flaticon](https://www.flaticon.com/search?word=christmas) (for icons)

**Note:** If images are not found, the app will automatically generate placeholder images using canvas.
