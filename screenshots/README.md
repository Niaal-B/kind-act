This directory contains screenshots of the Christmas Tree of Kindness application.

Add the following screenshots here:

2. `my-tree.png` - Personalized Christmas tree with decorations

## Tips

- Use PNG format for better quality
- Recommended dimensions: 1920x1080 or 1600x900
- Crop to show the most important UI elements
- Ensure text is readable
